================================================================================
COMO USAR
================================================================================

1. VERIFICAR DOCUMENTAÇÃO:
   cat BUGBOUNTY_CHEATSHEET_INTEGRATION.md

2. VER DEMOS INTERATIVA:
   python -m app.quickstart_cheatsheet

3. IMPORTAR EM SEUS SCANNERS:
   
   from app.payloads_integration import (
       get_xss_test_payloads,
       get_sqli_test_payloads,
       get_ssrf_test_payloads,
       get_lfi_test_payloads,
       get_redirect_test_payloads,
       get_rce_test_payloads,
   )

4. USAR OS PAYLOADS:
   
   # XSS
   xss_payloads = get_xss_test_payloads(bypass_waf=True)
   
   # SSRF com categoria específica
   aws_payloads = get_ssrf_test_payloads(target_type="aws_metadata")
   
   # Parâmetros comuns
   from app.payloads_integration import get_ssrf_parameters
   params = get_ssrf_parameters()

5. INTEGRAR COM SCANNER EXISTENTE:
   
   Ver arquivo app/example_ssrf_integration.py para exemplo completo
